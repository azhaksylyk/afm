# Автоматическая категоризация наименований ТРУ (без train.csv)

Проект реализует **полностью автоматическую категоризацию наименований ТРУ** (товары, работы, услуги) **без ручной разметки** и без отдельного `train.csv`.

Вместо классического обучения по размеченным данным используется подход:

- `SentenceTransformer` (BERT-подобная модель) → эмбеддинги предложений;
- `KMeans` → кластеризация наименований по смыслу;
- `TF-IDF` → извлечение ключевых терминов для каждого кластера;
- генерация **авто-имен категорий** на основе этих терминов.

На выходе мы получаем CSV-файл, где у каждого наименования есть:
- идентификатор кластера,
- автоматически сгенерированное название категории (`auto_category`),
- набор ключевых терминов кластера (`cluster_top_terms`).
